# Images for the project README

Add screenshots here so the root `README.md` can link to them with stable paths.

| File | Used in README |
|------|----------------|
| `preview.png` | Main preview image (web UI, CLI, or both — your choice) |

Use PNG or WebP. If you use a different name, update the `![…](…)` path in `README.md`.

**PyPI:** the project page renders `README.md` but does not serve repo files, so a relative image link may show as broken there. After the screenshot is on `main`, you can point `README.md` at a stable raw URL instead, for example  
`https://raw.githubusercontent.com/TheAsianFish/repolix/main/docs/images/preview.png`  
(or keep the relative path if you only care about GitHub).
